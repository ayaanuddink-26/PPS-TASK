This repository contains a collection of Python-based projects developed as part of the Programming for Problem-Solving Laboratory (ACSE07) course. These projects demonstrate the application of core programming concepts—such as functions, loops, and nested data structures—to solve real-world challenges in cybersecurity, agriculture, and commerce.📂 Projects Included1. Food Ordering SystemA console-based application designed to automate order management for small food businesses.Key Features: Menu display using dictionaries, automated bill calculation, and input validation for item IDs and quantities.Core Concepts: Dictionaries, user input handling, and loops.2. User Login Activity AnalyzerA cybersecurity tool that monitors login logs to identify potential security breaches.Key Features: Aggregates login attempts per user and flags accounts as "Suspicious" if they exceed three failed attempts.Core Concepts: Nested data structures, log aggregation, and batch processing logic.3. Weather-Based Harvest AdvisoryAn agricultural decision-support system that advises farmers on the optimal time to harvest based on short-term weather trends.Key Features: Analyzes 3–7 day trends for rainfall and humidity to mitigate post-harvest loss.Core Concepts: Data aggregation, numerical averaging, and conditional logic.🚀 Future EnhancementsTo address current system limitations, the following improvements are planned:Data Persistence: Integrating SQL or NoSQL databases to prevent data loss after program termination.Graphical User Interface (GUI): Moving beyond console-based inputs to improve user accessibility.Real-Time Data: Incorporating live APIs for weather feeds and payment gateways.📋 Course DetailsStudent Name: Mohd Khaja Ayaanuddin Roll Number: 25951A6671 Course: B.Tech CSE (AI & ML) Institution: Institute of Aeronautical Engineering (IARE)
